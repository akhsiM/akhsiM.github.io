General

Learning how to grab data from a JSON API using Python, then parse out the information we want and sort the data using a custom key.

https://www.youtube.com/watch?v=1lxrb_ezP-g

This tutorial looks at https://formulae.brew.sh/formula/.

What we are trying to do is to use the individual package's API to retrieve the analytic data. With this analytic data, we parse the "30d" parameter that contains the number of downloads for each package within 30 days and sort them using this data.

Practical Work

API that contains all packages: https://formulae.brew.sh/api/formula.json

Note that this API does not contain analytic information which we are looking at.

Prep Work

We start with importing the requests module, then importing the main API.

# First we import the modules
import requests
import json

# Then we retrieve the large API using requests.get
r = requests.get('https://formulae.brew.sh/api/formula.json')

# Then we parse the API through a large API object
packages_json = r.json()

We can verify the data by printing it. It should print out a huge chunk of text

print(packages_json)

We can clean it up a bit by using the json.dumps() method.

packages_str = json.dumps(packages_json, indent=2)
print(packages_str)

Looking at one package only

We can look at the first package only

packages_str = json.dumps(packages_json[0], indent=2)
print(packages_str)

We can also just grab the name of the first package only

package_name = json.dumps(packages_json[0]["name"])
print(package_name) ## This one should print "a2ps"

We can now look at individual package's API that should contain the analytic data that we want.

The API URL for the first package is this: https://formulae.brew.sh/api/formula/a2ps.json

Good, so it's in the format: https://formulae.brew.sh/api/formula/%7BPACKAGENAME%7D.json

Using the F-String, sub the parameter in to get what we want

We can now just grab the name of the package we want, then sub it into the URL using a F-String.

pckg_name = json.dumps(packages_json[0]["name"]).replace('"','')
url = f'https://formulae.brew.sh/api/formula/{pckg_name}.json'
print(url)
print(json.dumps(requests.get(url).json(), indent=2))

Look at analytics data only, after subbing the name parameter

pckg_name = json.dumps(packages_json[0]["name"]).replace('"','')
url = f'https://formulae.brew.sh/api/formula/{pckg_name}.json'

installs_30 = requests.get(url).json()['analytics']['install_on_request']['30d'][pckg_name]

installs_90 = requests.get(url).json()['analytics']['install_on_request']['90d'][pckg_name]

installs_365 = requests.get(url).json()['analytics']['install_on_request']['365d'][pckg_name]
print(pckg_name, installs_30, installs_90, installs_365)

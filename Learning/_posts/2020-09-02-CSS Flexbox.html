Learning

What is Flexbox?

CSS Flexbos is a layout mode that provides an easy and clean way to arrange items within a container. Since Flexbox, a newer CSS layout mode has come out, Grid, however they are different. Flexbox is essentially used for laying out items in a single dimensions (row OR column) however Grid is for layout of items in two dimentions (row AND column).

Flexbox is more responsive and mobile friendlier than the previous widely used payout system which is block. We can also use media query to make flexbox even better.

Flex container's margin do not collapse with the margins of its contents.

If we want to change the order of our column, we don't have to change the HTML but can simply assign order property to our flexbox.

Flexbox Box Model Concept

The general concept of flexbox model is the ability to alter item width and height to best fit in its containers available free space. Flexbox is also direction-agnostic i.e it works well for both vertical and horizontal direction.

Flexbox was created for small-scale layout while Grid is more for large scale

General Idea

Basically we have a main flex container wrapper with the display: flex property.

Inside this container we have child elements, so called flex items. There are certain properties that go on the containers and certain ones that go on the items themselves.

The x and y axis are called main and cross axis and we can use certain align property to position our flex items along these.


Flex Properties

  • display: flex | inline-flex;

This goes on the container element.

  • flex-direction: row | column;
  • flex-wrap: wrap | nowrap | wrapreverse
  • flex-basis: <length>

This is pretty much the same as width which can be used to assign a width to our flex item.

  • justify-content: flex-start | flex-end | center

This is used to align item to left (flex-start), right (flex-end) or center.

  • align-self: flex-start | flex-end | center.

This allows us to overwrite default alignment for individual flex items. This is used to specify the alignment for selected flex items inside a flex container. It overrides the align-items property.

  • align-items

This property is used to set the default behaviour for how flex items are laid out across main axis.

  • align-content

Similar to align-tems but for cross axis.

  • flex-grow: <number>
  • flex-shrink: <number>

These are used to make different elements in the row different sizes, for example if we want the main column to take different amount of space as a nav column.

  • flex: <int> can be used for both flex-grow and flex-shrink together.
  • order: <int> can be used to change order of element without changing the HTML.

Some Practice

Starting Out

Suppose we start out with this:

<style>
  .container-1 {
  }
  
  .container-1 div {
    border: 1px #ccc solid;
    padding: 10px;
  }
</style>

<div class="container-1">
  <div class="box-1">
    <h3>Box One</h3>
  </div>
  <div class="box-2">
    <h3>Box Two</h3>
    <p>Random Text in box two</p>
  </div>
  <div class="box-3">
    <h3>Box Three</h3>
    <p>Random Text in box three</p>
  </div>
</div>


Make elements appear side by side

By simply declaring display: flex in the container element, i.e:

<style>
  .container-1 {
    display: flex;
  }
  
  .container-1 div {
    border: 1px #ccc solid;
    padding: 10px;
  }
</style>

<div class="container-1">
  <div class="box-1">
    <h3>Box One</h3>
  </div>
  <div class="box-2">
    <h3>Box Two</h3>
    <p>Random Text in box two</p>
  </div>
  <div class="box-3">
    <h3>Box Three</h3>
    <p>Random Text in box three</p>
  </div>
</div>

We'll get the flex items side-by-side:


Give all elements the same size with flex

We can then add flex: 1 to all our flex items to give them the same size:

<style>
  .container-1 {
    display: flex;
  }
  
  .container-1 div {
    border: 1px #ccc solid;
    padding: 10px;
  }
  
  .box-1, .box-2, .box-3 {
    flex: 1;
  }
</style>

<div class="container-1">
  <div class="box-1">
    <h3>Box One</h3>
  </div>
  <div class="box-2">
    <h3>Box Two</h3>
    <p>Random Text in box two</p>
  </div>
  <div class="box-3">
    <h3>Box Three</h3>
    <p>Random Text in box three</p>
  </div>
</div>


Or, if we want to make box one bigger than the other:

align-items

By default, flex items within a container are stretched to the same size.

We can add the align-items property to the container element to adjust the item according to their content:

<style>
  .container-1 {
    display: flex;
    align-items: flex-start;
  }
  
  .container-1 div {
    border: 1px #ccc solid;
    padding: 10px;
  }
  
  .box-1 {
    flex: 2;
    order: 2
  }
  
  .box-2 {
    flex: 1;
    order: 1;
  } 
  
  .box-3 {
    flex: 1;
    order: 3;
  }
</style>

<div class="container-1">
  <div class="box-1">
    <h3>Box One</h3>
  </div>
  <div class="box-2">
    <h3>Box Two</h3>
    <p>Random Text in box two</p>
  </div>
  <div class="box-3">
    <h3>Box Three</h3>
    <p>Random Text in box three</p>
  </div>
</div>

justify-content

With the specified width in container-2, we notice a gap on the right hand side of it. Let's add a margin in.

The default setting for justify-content is flex-start. If we change it to center, i.e:

.container-2 {
  display: flex;
  justify-content: center;
}

We'd see this:


If we use flex-end, the container will be pushed over to the right side.

However, if we want our containers to be aligned with each other vertically, we'd use space-between:

Using media query

To make our page more responsive, we can put container-1 and container-2 within a media query like this:

<style>
  @media(min-width: 768px){
    .container-1 {
    display: flex;

   }
  
    .container-2 {
    display: flex;
    justify-content: space-between;
   }
  }

  
  .container-1 div,
  .container-2 div {
    border: 1px #ccc solid;
    padding: 10px;

  }
  
  .container-2 div {
    flex-basis: 30%
